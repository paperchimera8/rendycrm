Политика конфиденциальности

Политика конфиденциальности сервиса «РЕНДИСРМ» 1. Общие положения 1.1. Настоящая Политика конфиденциальности определяет порядок обработки и защиты персональных данных пользователей сервиса «РЕНДИСРМ» (далее — «Сервис»). 1.2. Используя Сервис, Пользователь выражает согласие с настоящей Политикой. 2. Какие данные мы собираем 2.1. Данные, предоставляемые при регистрации: * имя, * номер телефона, * адрес электронной почты, * имя пользователя в Телеграм. 2.2. Данные клиентов Пользователя, которые загружаются в CRM: * имя клиента, * телефон, * email, * история записей, * переписка, * иная информация, добавленная Пользователем. 2.3. Технические данные: * IP-адрес, * данные браузера, 3. Цели обработки данных Персональные данные обрабатываются для: * предоставления доступа к Сервису; * обработки заявок и записи клиентов; * работы AI-ассистента; * улучшения качества Сервиса; * исполнения требований законодательства. 4. Правовые основания обработки Обработка осуществляется на основании: * согласия субъекта персональных данных; * исполнения договора (пользовательского соглашения); * требований законодательства. 5. Передача данных третьим лицам 5.1. Данные могут передаваться: * платежным системам; * хостинг-провайдерам; * сервисам аналитики; * подрядчикам, обеспечивающим работу AI. 5.2. Передача осуществляется только в объеме, необходимом для работы Сервиса. 6. Хранение данных 6.1. Данные хранятся в течение срока действия учетной записи. 6.2. По запросу Пользователя данные могут быть удалены. 7. Защита данных Компания принимает разумные технические и организационные меры для защиты данных от: * несанкционированного доступа, * изменения, * уничтожения, * распространения. 8. Права пользователя Пользователь вправе: * получать информацию о своих данных; * требовать их исправления; * требовать удаления; * отозвать согласие на обработку.
